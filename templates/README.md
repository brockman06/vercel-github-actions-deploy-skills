# Templates

Comprehensive guides for setting up Vercel deployments via GitHub Actions.

## Available Templates

| Template | Description |
|----------|-------------|
| [deploy-workflow-template.md](deploy-workflow-template.md) | Full step-by-step setup guide with advanced configurations, PR previews, monorepo support, and troubleshooting |

## When to Use Templates

- **First-time setup**: Follow the complete guide in `deploy-workflow-template.md`
- **Quick setup**: Use the ready-to-copy workflow files in `examples/` instead
- **Advanced use cases**: Templates cover PR previews, monorepos, Slack notifications, and path-based triggers
